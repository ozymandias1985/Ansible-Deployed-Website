About This Project

A beginner-friendly Ansible project demonstrating real-world infrastructure automation on Fedora/RHEL systems.

This project automates the complete deployment of a web server, from initial system setup to final website deployment. It's designed to teach Ansible fundamentals while working with enterprise Linux systems (Fedora, RHEL, CentOS).
